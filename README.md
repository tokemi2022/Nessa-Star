# ⭐ Star Girl — Nessa's Development Journey

A personal child development tracker and AI-powered activity planner for Nessa, built for two working parents in Barcelona.

## Features

- 🤖 **AI-generated weekly plans** — 5 activities per week, tailored to Nessa's age, languages, and developmental domains
- ✅ **Simple logging** — Parents tap Done or Skip. No manual data entry
- 🌟 **Milestone tracking** — Log and celebrate every first
- 📸 **Moments gallery** — Photos and notes from both parents
- 📊 **Progress charts** — Domain progress across 7 development areas
- 🌍 **4-language awareness** — English, Yoruba, Spanish, Catalan

## 7 Development Domains

| Domain | Focus |
|--------|-------|
| 🧠 Cognitive | Problem-solving, memory, patterns |
| 🗣️ Language & Signs | 4 languages + sign language |
| ❤️ Emotional Intelligence | Self-regulation, empathy |
| 🤸 Physical & Motor | Fine/gross motor skills |
| 🎨 Creativity & Arts | Imagination, divergent thinking |
| 🤝 Social & Character | Values, kindness, leadership |
| 🌍 Cultural Identity | Yoruba heritage + European context |

## Hosting on GitHub Pages

1. Create a new GitHub repository (e.g. `nessa-littlestar`)
2. Upload `index.html`, `style.css`, and `app.js`
3. Go to **Settings → Pages**
4. Set source to **Deploy from a branch → main → / (root)**
5. Your app will be live at `https://yourusername.github.io/nessa-littlestar`

Both parents can bookmark the URL on their phones and access it like a native app.

## Adding to Phone Home Screen

**iPhone:** Open in Safari → Share → Add to Home Screen  
**Android:** Open in Chrome → Menu → Add to Home Screen
